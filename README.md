# popx Login Interface

> A modern, responsive login and profile management interface built with React, Tailwind CSS, and localStorage-based authentication. This project demonstrates a full authentication flow (login, registration, profile, password reset) with a beautiful UI and no backend required.

## 🚀 Features

- **User Authentication**: Register, login, logout, and update profile, all managed in the browser with localStorage.
- **Protected Routes**: Profile and edit-profile pages are accessible only to authenticated users.
- **Responsive UI**: Clean, mobile-friendly design using Tailwind CSS and custom themes.
- **Profile Management**: Users can update their name, email, about, and profile image.
- **No Backend Needed**: All data is stored locally for easy demo and testing.

## 🛠️ Technologies Used

- React (with hooks)
- React Router DOM
- Tailwind CSS (with plugins: forms, container-queries)
- localStorage API

## 📁 Project Structure

- `src/pages/` – Home, Login, CreateAccount, ForgotPassword, Profile, EditProfile
- `src/context/AuthContext.js` – Authentication context and logic
- `src/utils/storage.js` – LocalStorage utility functions
- `src/App.js` – Routing and protected route logic
- `tailwind.config.js` – Custom theme and Tailwind plugins

## 🖥️ Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start the development server:**
   ```bash
   npm start
   ```
3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📦 Available Scripts

- `npm start` – Run the app in development mode
- `npm run build` – Build for production
- `npm test` – Run tests
